```

│ Thunder -16.5  →  雷霆赢 17+ 分才赢                          │

| 雷霆赢 16 分 | ✅ 赢 | ❌ 输 | ❌ 输 | 小赚 |
| 雷霆赢 7-15 分 | ❌ 输 | ❌ 输 | ❌ 输 | 全输 |

| 活塞赢 | ❌ 输 | ❌ 输 | ❌ 输 | 全输 |

• 在同一市场同时持有 Yes 和 No
• 两边仓位金额相等
• 无论结果如何，一边赢一边输，总收益接近 0
• 但可以通过买卖价差赚取利润

做市原理:
┌─────────────────────────────────────────────────────────────┐
│ 假设 bid-ask spread = 2%                                    │
│ 买入 Yes @ 0.49, 卖出 Yes @ 0.51                            │
│ 买入 No @ 0.49, 卖出 No @ 0.51                              │
│                                                             │
│ 每 $100 交易量赚取 $2 价差                                   │
│ $1.4M 仓位可能已赚取可观的价差收益                            │
└─────────────────────────────────────────────────────────────┘
```
